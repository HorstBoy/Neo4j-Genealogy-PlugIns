Preparing Your Curated Files

Your genealogy data needs to be prepared for this software to import it into Neo4j. This software and the Neo4j PlugIn user defined functions will take your GEDCOM file and create nodes and edges in Neo4j. You will need to have the directory and file name of your GEDCOM.

Linking your family tree to kit DNA results

Open your GEDCOM File and the FTDNA root directory. Retrieve the Excel file from the link in the window to the right.
Enter the Match_Name, Curated_RN and Kit where known. Leave fields empty if not known

Triangulation Groups

Retrieve the Excel file from the link in the window to the right. Enter known triangulation group data.
