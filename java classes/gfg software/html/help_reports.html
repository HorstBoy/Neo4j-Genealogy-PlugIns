Reports

Reports are run using queries in the Neo4j broswer which can be opened from the Neo4j Desktop or in your browser with this URL: http://localhost:7474/browser/. While we could automate this with clicks in this application and may do so it future versions, there are advantages of you entering them in the Neo4j browser. You will see how the queries are written. You will be able to customize the results by the parameters you enter. But don't worry, we will provide you the information and ability to cut and paste queries into the Neo4j Browser. As you gain familiarity with Neo4j queries fell free to create some of your own and experiment. The links below or the Report menu items will take you to each report.    We are still writing the documentation so many links below are not yet active.
