Starting a New Project

This page guides you through starting a new project. It's quite easy:
  1. Enter a name of the project. Make it short and meaningful. It will be used in next steps.
  2. Enter a description to remind you of it purpose.
  3. Click the "Add Project" buttom and it will be saved and you will be directed to the next step.
