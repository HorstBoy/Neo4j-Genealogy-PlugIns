Project Management

GFG software supports multiple projects such as distinct projects for branches of your family tree or for different clients. Each project has a distinct directory with the relevant files. You start an initial or add a new project from the menu: Project ➔ New Project. Enter the project name and a description which will help you remember its purpose. Click "Add Project" and you will be taken to the project setup page. You can also get to this page to edit and existing project from the menu: Projects ➔ Open Project. Select the project in the drop down box. You will then see the fields required by the software to identify the database and locate the files. Use the text buttons to the right of each field to get information or navigate to the folder or file. Save or update the information you entered or edited.

Finally, you must activate the project. This writes a file, which will be displayed, that the software uses in performing its tasks.

You are now ready to load and analyze your data!
