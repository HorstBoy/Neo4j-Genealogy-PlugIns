Updating and Activating Your Project Parameters

This page helps you curate the information needed to load your genealogy family tree and DNA data. The software needs to know some paramenters including where your data is located. Here are the steps you need to take:
  1. Select the project from the combobox
  2. Fill in the required information. If the entry is for a folder or file, use the link adjacent to the textbox to navigate to the desired location.
    1. user database: the name you selected when creating the Neo4j database for this project.
    2. neo4j username: defaults to neo4j, but you might have change this when you first created your project when installing Neo4j
    3. neo4j password: the password you selected when you first created your Neo4j project. It will be the same for each database created within the project.
    4. neo4j import directory: you can navigate to this directory or paste it into the text box. You can find this folder by going to the Neo4j desktop, and navigating from the ellipsis: "...." ➔ Open Folder ➔ Import (click).
    5. GEDCOM file: navigate to the file. The entry should be the full path with the file name.
    6. FTDNA root directory: navigate to the folder. This directory will contain subdirectories and their files which the software will read in uploading your data to Neo4j
    7. curated rn and gedcom file: navigate to the file you prepared. The recommended and default location is the FTDNA root directory, but you can keep this anywhere.
    8. tg file: navigate to the file you prepared. This contains the triangulation groups and their boundaries.
  3. There are three actions you can initiate
    1. Save the data: this is available the first time you create the project
    2. Update the data: used when you edit the entries
    3. Activate the project: this writes a file for Neo4j PlugIn to use in processing your data. This must be activated before you run the processing and if you change projects. This file is saved is a specific directory and should not be changed.
Once you've completed these task, you are ready to process your data. This is done using the links under the Load Data menu item.
