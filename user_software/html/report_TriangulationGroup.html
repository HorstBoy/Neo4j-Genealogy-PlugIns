Triangulation Group Reports

Background

Triangulation groups are defined as chromosome segments shared by descendants of a common ancestor. These resports incorporate phasing, which limits the common ancestor to a single specific ancestor. Some other applications report triangulation groups to an ancestor couple, which can create ambiguities if there are half siblings amoung their descendants.

Data Requirements

The GEDCOM file provides the family tree information used.
The FTDNA chromosome browser file provides segment nodes and match-segment edges.

The curated triangulation group file provides the triangualtion group broundaries.
