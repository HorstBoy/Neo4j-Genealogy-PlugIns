Installing Neo4j

Before using this software you will need to install the Neo4j database. This panel will guide you through that process.
  1. Install Neo4j Enterprise Edition, version 4.x. The process starts here.
  2. When you first open the software you see the "home page." You do not yet have your database.
  3. From the home page you create a project with a name of your chosing. A project can hold numerous databases which might support projects on several family lines.
  4. Next, open the Neo4j browser. Their are two options
    1. Click the Open button in Neo4j Desktop (see figure below)
    2. In your browser, using this url: http://localhost:7474/browser/
  5. The browser will ask you to login. The user name is neo4j. The initial password is also neo4j. But you will be asked to enter your own password; do not forget it!
  6. Create your first database by opening the Neo4j browser, selecting the system database and typing this command: create database {name} {your database name}. The name must be only lower letters. You can create other databases by repeating this command. To delete a database type drop database {name}.
  7. Verify that you can start and stop the project database from the home page. This is important because the next steps may damage the functionality and you'd like to learn where this happened. Thus, be careful with this step.
  8. Modify the neo4j configuration file to accomodate the project requirements.
    1. Open the configuration folder. The Neo4j application opens to a "home page" from which you can open the Configuration folder. The link is in the "...." icon to the right of your project "Open" button. The navigation is "...." ➔ Open Folder ➔ Configuration.

      Neo4j Desktop navigation is in the upper right corner. From here you can Open the Neo4j Browser or navigate to the Import, PlugIn and Configuration folders.

    2. Open the file neo4j.conf in a text editor (Notepad or Notepad++; not Word or other tools that add extraneous text). Save this original file in case something goes awry and you need to revert to it.
    3. At the bottom of the config file paste these lines:
      apoc.export.file.enabled=true
      apoc.import.file.enabled=true
      dbms.security.procedures.unrestricted=jwt.security.*,apoc.*,gds.*,gen.*
      dbms.security.procedures.allowlist=jwt.security.*,gds.*,apoc.*, gen.*
      dbms.checkpoint.interval.time=30s
      dbms.checkpoint.interval.tx=1
      dbms.tx_log.rotation.retention_policy=false
      dbms.tx_log.rotation.size=1M
      dbms.transaction.timeout=30m
    4. Search for "dbms.security.procedures.unrestricted" to locate entries above those just entered and comment out the lines by adding a # at the start of the line.
    5. Search for "dbms.memory.heap." and adjust the memory upward to speed the processing. The number you select will depend on the RAM you have installed. I am a power user with lots of RAM and set as follows
      dbms.memory.heap.initial_size=1G
      dbms.memory.heap.max_size=4G
Install the Neo4j Genealogy PlugIn

Intalling the Neo4j Genealogy PlugIn enabled functions useful to genealogists. You can download the most recent plugin from GitHub. The plugin is a *.jar file. Save it in the Neo4j PlugIn directory. The Neo4j application opens to a "home page" from which you can open the PlugIn folder. The link is in the "...." icon to the right of your project "Open" button. The navigation is "...." ➔ Open Folder ➔ Plugins. Copy the jar file into that folder and then restart the database. Verify the installation by opening the Neo4j browser and entering this command which will show you a list of functions available:
Show Functions yield name, signature, description,returnDescription,aggregating where name STARTS WITH 'gen' return name, signature, description,returnDescription,aggregating
Create your first Neo4j Database

In the Neo4j Browser, select the system database and the enter this command:
create database {your name}
Neo4j is now ready to receive your data! If, in your initial experiments, you need to start over, you can delete this database and re-created in. To delete a database use this command:
drop database {your database}
